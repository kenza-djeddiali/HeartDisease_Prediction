# Heart Disease Risk Prediction App

Le projet **Heart Disease Risk Prediction** utilise des techniques d'apprentissage automatique pour prédire si un individu risque de développer une maladie cardiaque en fonction de certaines caractéristiques médicales et biologiques. L'application Web permet à l'utilisateur de remplir un formulaire avec ses données de santé, puis d'obtenir une prédiction de risque de maladie cardiaque, avec un graphique représentant le risque.

Le backend est développé avec **FastAPI** pour gérer les requêtes API et utiliser un modèle de Machine Learning pour effectuer les prédictions. Le frontend est développé avec **React** pour fournir une interface utilisateur interactive et responsive.


## Prérequis

### Backend (Python)
- Python 3.6 ou version supérieure
- FastAPI
- Uvicorn
- joblib
- scikit-learn

### Frontend (JavaScript)
- Node.js
- npm ou yarn
- React
- React Bootstrap
- axios
